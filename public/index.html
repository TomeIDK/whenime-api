Whenime API

User Routes

GET /users - Get all users GET /users/{id} - Get user by ID POST /users - Add a user PATCH /users/{id} - Update user by ID DELETE /users/{id} - Delete user by ID

News Routes

GET /news - Get all news GET /news/{id} - Get news by ID POST /news - Add news item PATCH /news/{id} - Update news item by ID DELETE /news/{id} - Delete news item by ID

Contact Form Routes

GET /forms - Get all contact forms GET /forms/{id} - Get form by ID POST /forms - Add contact form submission PATCH /forms/{id} - Update form by ID DELETE /forms/{id} - Delete form by ID
