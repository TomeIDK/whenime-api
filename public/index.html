Whenime API

User Routes

GET /users - Get all users GET /users/{id} - Get user by ID

News Routes

GET /news - Get all news GET /news/{id} - Get news by ID

FAQ Routes

GET /faq - Get all faq categories & questions GET /faq/categories - Get all faq categories GET /faq/questions - Get all faq questions GET /faq/categories/{id} - Get faq category by ID GET /faq/questions/{id} - Get faq questions by ID

Contact Form Routes

GET /forms - Get all contact forms GET /forms/{id} - Get form by ID
